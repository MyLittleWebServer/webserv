Detailed Description

서버를 관리하는 클래스

Kqueue 객체 _eventQueue를 갖고있습니다. ServerManager는 Config를 초기화하고, Server를 초기화합니다. 그리고 Server를 시작하고, Server 시작을 prompt에 알립니다.

◆ initServer()

void ServerManager::initServer ( void  )

initServer 함수는 _listenOrganizer에 저장된 포트를 기반으로 서버를 초기화합니다.

서버를 초기화하는 과정은 다음과 같습니다.

  1. Server 객체를 생성합니다.
  2. Server 객체의 initServerSocket() 함수를 호출합니다.
  3. Server 객체의 getSocket() 함수를 호출하여 반환된 소켓 디스크립터를 이벤트 큐에 추가합니다.
  4. Server 객체의 getSocket() 함수를 호출하여 반환된 소켓 디스크립터를 Kqueue 클래스의 setFdSet() 함수를 호출하여 FD_SERVER 타입으로 설정합니다.
  5. Server 객체를 _serverVector에 저장합니다.

void ServerManager::startServer ( void  )

startServer 함수는 _serverVector에 저장된 서버들을 시작합니다.

EventHandler 객체를 생성합니다. EventHandler 객체의 setCurrentEvent() 함수를 호출하여 현재 이벤트를 설정합니다. EventHandler 객체의 checkFlags() 함수를 호출하여 현재 이벤트의 플래그를 확인합니다. EventHandler 객체의 branchCondition() 함수를 호출하여 현재 이벤트의 플래그에 따라 분기합니다.
